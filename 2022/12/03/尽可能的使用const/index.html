尽可能的使用const

发表于2022-12-03|更新于2023-01-10|effectiveC++
|阅读量:

const 语法的使用

  1. 如果关键字const出现在*左边表示所指之物为常量
  2. 如果关键字出现在*右边，表示指针自身式常量
  3. 如果出现在*两边，表示被指物和指针两者都是常量

const 成员函数

  1. 优点

    1. 使得class的接口比较容易理解，可以得知那个函数可以改动对象，哪个函数不行
    2. 他们使得 ==操作const 对象成为可能==

经验之谈

==令函数返回一个常量值，往往可以降低因客户错误而造成的意外，而又不至于放弃安全性和高效性==

举例说明

当用户在这里想进行比较的时候，而进行了一个赋值操作，如果不加const 则这个操作则不会产生错误，而我们自己写的类与内置类型要保持一致，所以在这里我们需要加上 ==const==加以限制
